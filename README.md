# Silicon Labs IoT Training Guides

Step-by-step guides for building IoT projects with Silicon Labs hardware, Home Assistant, and Matter/Thread.

## Available Guides

### [Matter Thermal Mesh](./matter-thermal-mesh/)
Build a whole-home thermal presence sensor network using:
- **SparkFun Thing Plus Matter** + Grid-EYE thermal sensor
- **Arduino Nano Matter** + LED matrix heatmap display
- **XIAO MG24** as Thread Border Router
- **Home Assistant** for automation
- **Custom Flask + D3.js dashboard** on RPi 7" touchscreen
